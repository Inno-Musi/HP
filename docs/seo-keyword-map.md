## 0. 現状診断（なぜ流入が少ないか）

GSC実測10日分:

| 指標 | 値 | 読み方 |
|---|---|---|
| 表示回数 | 514 | 致命的に小さい。そもそも検索結果に露出していない |
| クリック | 78 / CTR 15.2% / 平均順位 9.7 | 出れば押される・順位も悪くない=技術/CTRは問題でない |
| インデックス | 登録49 / 未登録46 | 主要ページは登録済み。インデックスは主因でない |

**56クエリのほぼ全部がブランド名検索**（musico / 株式会社musico / ミュジコ / 石黒啓介 等）。
残りは `sisuco` `クラシコ` `ナデシコ` 等、Googleが音で誤マッチした別会社。
**見込み客が課題で打つ非ブランド語の表示は実質ゼロ。** 唯一の芽 = `役員専用食堂`(1表示/順位6)、`musico ai`、`役員専用食堂`。

→ 原因は技術ではなく **「買い手の検索意図に合うコンテンツ露出」と「新規ドメインの権威不足」**。土台(schema/sitemap/hreflang/CWV/GA4)は完成済み。ここからは中身と内部リンクと権威。

## 5. AIO（AI検索エンジン最適化）チェックリスト

SEOと根は同じ（実在の質問に構造化して答える）。加えて:

- [ ] `/llms.txt` を主要サービス＋insightsの要約付きインデックスに拡充（現状は最小限。middleware除外はPR#23で対応済）
- [ ] 各記事の冒頭に**40〜60字の定義/結論文**（AIが抜き出す単位）。H2は疑問文で。
- [ ] FAQPage schema をPAA語（役員食堂とは/直営と委託どちら/費用相場 等）に一致させる
- [ ] 第三者言及を増やす（AIは複数ソースで裏が取れる主体を引用しやすい）: プレスリリース、クライアント/パートナーサイトからの被リンク、業界メディア寄稿、代表・石黒のLinkedIn/note からの記事リンク
- [ ] 検証: ChatGPT/Perplexity/Google AI Overviews で「役員食堂 運営会社」「外資系 オフィスカフェ 運営」等を実際に聞き、MUSICOが出るか毎月チェック

## 6. 権威づけ（オフページ／新規ドメインの最重要課題）

被リンクほぼゼロが競合語で上がれない根本原因。優先:

1. クライアント/パートナー(GSS等)サイトからの被リンク依頼
2. プレスリリース配信（PR TIMES等）で事例・提携を発信＝一次被リンク＆AIO言及源
3. 代表・役員の LinkedIn/X/note のプロフィール＆投稿からサイト/記事へリンク（UTM付）
4. Wantedly/採用媒体の会社紹介からコーポレートサイトへ
5. 業界ディレクトリ・登壇・寄稿
